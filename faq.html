How to Create

On the main Page there is a plus button, click that and a menu will slide down at the top of the list. From there you can fill in the information for your to-do item.

How is my list saved?

In the settings there is an option to change the how your to-do list is saved or to not have it saved.

Local Storage (Default): This option saves the list to the local storage within the user's web browser. This allows the user to exit the browser and when they come back to the site the list will still be there.
Session Storage: This option saves the list to the session storage within the user's web browser. The list is only saved while the browser remains open. When the browser is closed all data will be erased.
Don't Save: This does not save the list at all and closing the tab or even simplying changing to a different page within the website may erase the data. (Not Recommended)

Notice - This website does not store the data regarding the users to-do list or preferences on any server, instead all of that information and data is saved locally.
